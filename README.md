## Features

- [x] SSG 静态渲染， SEO 友好，极速渲染
- [x] 生成站点地图( @astrojs/sitemap )
- [x] 优秀的 [Lighthouse](https://web.dev/measure/) 和 [PageSpeed Insights](https://pagespeed.web.dev/) 报告分数
- [x] 支持 RSS 订阅
- [x] 使用 tailwindcss 构建样式
- [x] 使用 [Vercel](https://vercel.com/) 部署
- [x] 移动端响应式布局
- [x] 支持代码高亮
- [x] 支持文章分类、标签
- [x] Markdown MDX 支持
